#  Процедура запуска
## На клиенте: (Терминал 1)
cd client

# Запуск клиента на localhost:8080
npm run dev


## На сервере (Терминал 2)
cd server

# Запуск сервера на localhost:5000
npm run dev

После запуска - http://localhost:8080/
